Introducción a HTML5 y CSS3

HTML5

¿Qué significa HTML?

HTML es el acrónimo en ingles de HyperText Markup Language, que en español se traduciría como Lenguaje de Marcado de Hipertexto.

¿Qué es HTML?

HTML es un lenguaje de marcado interpretado por el navegador del cliente.

El lenguaje se compone principalmente de etiquetas y atributos, lo cual facilita tanto su comprensión a la hora de aprenderlo, como su lectura a la hora de revisar el código.

¿Para que se utiliza HTML?

El uso principal de HTML es el de dar una estructura a la página.

Breve historia sobre HTML

HTML apareció gracias a la necesidad de compartir documentos entre diferentes organizaciones sin tener que adaptar de forma significativa la información para que esta fuese accesible desde diferentes ordenadores.

A principios de los 90 Tim Berners-Lee publicó la primera descripción oficial de HTML, lo que marcó el inicio del desarrollo de HTML y de la WWW que serían mantenidos por el W3C.

En el año 1998 fué cuando se implanto junto a la versión 4.0 de HTML el uso de las hojas de estilo o CSS y la capacidad de ejecutar scripts del lado del cliente.

Actualmente la organización que mantiene el lenguaje se llama WHATWG la cual está compuesta por integrantes de Apple, la Fundación Mozilla y Opera Software.

NOTA! Al ser un lenguaje interpretado de lado del cliente, siempre debemos tener en cuenta pueden haber navegadores que no reconozcan ciertas propiedades de css, lo que genera fallos inesperados al cargar la web en esos navegadores.

Creación de un archivo HTML

Para crear un archivo HTML solo necesitamos un editor de texto cualquiera.

A continuación se expondrán el proceso de creación de un documento HTML en orden:

  1. Crear un documento de texto nuevo
  2. Añadir las etiquetas necesarias y opcionales en caso de haberlas
  3. A la hora de guardar el documento, pulsaremos en guardar como
  4. y cuando se muestre el nombre y tipo de archivo, en la sección tipo elegiremos todos los archivos
  5. finalmente en la sección nombre elegiremos el nombre de nuestro archivo pero al final añadiremos .html
  6. pulsamos en guardar y listo.
  7. En caso de querer modificar el archivo solamente tenemos que hacer click derecho sobre el archivo y abrirlo con un editor de texto. Cuando hayas terminado de modificarlo solo vuelve a repetir los pasos anteriores en orden.

Todo archivo HTML se compondrá como mínimo de: 


    <!DOCTYPE html>
        <html lang="es-es">

          <head>
              <meta charset="utf-8">
              <meta 
                  name="viewport" 
                  content="
                    width=device-width, 
                    initial-scale=1.0">
              <title>Titulo Pestaña</title>
          </head>
 
          <body>
             <!--Recomendable añadir header-->      
              <header>
              </header>
                
             <!--Recomendable añadir main-->
              <main>
              </main>
                
             <!--Recomendable añadir footer-->
              <footer>
              </footer>
          </body>

        </html>

Significado de cada una de las etiquetas y atributos

  1. !DOCTYPE: La etiqueta !DOCTYPE sirve para definir el tipo de documento que vamos a utilizar, añadiendo al lado derecho el valor correspondiente. En este caso html.
  2. html: La siguiente etiqueta, html es la que alojará todo el contenido que vamos a definir.
    • El atributo lang="es-es", define el idioma principal del documento/página. En este caso se está usando es-es porque queremos que el idioma principal sea el castellano y el pais España, pero se puede cambiar por el idioma y pais que se desee en caso de estar usando otro dialecto o idioma. Lo único que el valor deberá estar en formato ISO 639-1 como en el ejemplo.
  3. head: Aquí se agruparán todas las etiquetas de "configuración" y carga de archivos como pueden ser las meta etiquetas y las hojas de estilos entre otras.
  4. meta: Las meta etiquetas son las encargadas de definir la mayor parte de los "ajustes" de la página, estas etiquetas siempre irán acompañadas de algún atributo o valor. Algunos atributos pueden ir acompañados de un nombre(name=) que será el encargado de definir el uso del atributo.
    • El atributo charset="utf-8", es el encargado de establecer el tipo de codificación que usará la página/documento. En este caso estamos haciendo uso de la codificación utf-8. Esta etiqueta deberá ser la primera que se añada dentro de la etiqueta head.
    • El atributo content="width=device-width, initial-scale=1.0" irá siempre acompañado del nombre name="viewport" al principio. Sirve para que el navegador cargue la página en base al tamaño de la pantalla y con un zoom del 100%. Aunque se pueden cambiar los valores no es lo aconsejable.
  5. title: El texto que contenga la etiqueta title es el que se mostrará en la pestaña del navegador y en los resultados de búsqueda. Lo ideal es que el texto tenga entre 40 a 70 caracteres como máximo.
  6. body: Aquí se agrupará todo el contenido que queremos mostrar en nuestro documento o página.
  7. De forma opcional pero muy recomendable, se pueden añadir las siguientes etiquetas para que el documento tenga una semantica perfecta:
    • header: Dentro del header normalmente se incluye el logotipo, la barra de navegación, el título de la página y elementos similares.
    • main: La etiqueta main agrupará el contenido principal de la página o documento.
    • footer: Finalmente la etiqueta footer normalmente agrupa elementos como los links de contacto, los derechos de la página, etc.

Cuando se quiere hacer uso de los comentarios dentro del código podemos encerrar el texto o código dentro de las etiquetas <!-- comentario --> , podemos hacer uso de cuantos comentarios queramos pero cuidado con comentar elementos sensibles ya que todo lo que se escriba se mostrará en la consola del navegador.

NOTA! Las etiquetas mostradas en este apartado no se deben de repetir dentro de un mismo documento.

NOTA! Muy importante el orden de dispocisión de las etiquetas deberá ser igual al expuesto recientemente.

CSS3

¿Qué significa CSS?

CSS es la abreviatura de Cascading Style Sheet que en castellano significa Hojas de Estilo en Cascada

¿Qué es CSS y para qué se utiliza?

CSS es un lenguaje de estilos en cascada, lo que significa que se ejecuta desde arriba hacia abajo, sobreescribiendo las declaraciones anteriores en caso de que ambas tengan la misma preferencia. CSS se utiliza para dar forma y color a las distintas etiquetas HTML que hayamos definido.

CSS está compuesto principalmente de selectores y reglas que agrupan las distintas declaraciones.

Significado de:

  • Declaraciones: Las declaraciones son las instrucciones que van a definir como se debe representar un elemento especifico.
    Todas las declaraciones se compondrán de dos elementos:
    1. La propiedad: background, color, font...
    2. El valor: red, cyan, 10px...
    Todas las declaraciones separarán las propiedades de los valores mediante el uso de dos puntos y siempre se dará por finalizada una declaración cuando se añada un punto y coma al final de la declaración:
    • propiedad : valor ;
    • Ejemplo: background-color : cyan ;
  • reglas: Una regla es la agrupación de una o varias declaraciones dentro de unas llaves: 
      
    { declaraciones }

    {
        background-color : lightgray ;
        font-size : 20px ;
        font-family : Verdana, Arial, "Sans-serif" ;
    }
  • Selectores: Los selectores son los elementos sobre los que queremos aplicar las reglas. Entre los diferentes selectores que hay los más utilizados son:
    1. El nombre del elemento HTML: p, ul, h1...
      Se utiliza cuando queremos aplicar un conjunto de estilos a todos los elementos HTML que tienen el mismo nombre. Los estilos que definamos haciendo uso del nombre del elemento HTML se aplicarán solos automáticamente. 
      
    p  { 
          font-size : 20px ;
    }
    body  { 
          background-color : lightgoldenrodyellow ; 
    }
    2. Un punto: .
      Irá acompañado de un nombre personalizado a su derecha, se suele utilizar cuando queremos aplicar un conjunto de estilos a un número determinado de elementos HTML. El nombre que elijamos es el que usaremos como valor dentro del atributo class="", el cual situaremos dentro de la etiqueta de apertura del elemento al que le vayamos a aplicar el estilo.
      A este selector se le conoce como clase. 
      
    .nombre_para_tu_clase  { 
            font-size : 20px ;
    }
    .fondo_gris  { 
            background-color : gray ; 
    }
    3. Una almohadilla: #
      Irá acompañada de un nombre personalizado a su derecha, se suele utilizar cuando queremos aplicar un conjunto de estilos a solo un elemento HTML. El nombre que elijamos es el que usaremos como valor dentro del atributo id="", el cual situaremos dentro de la etiqueta de apertura del elemento al que le vayamos a aplicar el estilo.
      A este selector se le conoce como id. 
          #nombre_para_tu_id  { 
            font-size : 20px ;
    }
    #estilo_header  {  
        width : 100% ;
        height : auto ;
        background-color : orange ;
    }

  7. En caso de querer modificar el archivo solamente tenemos que hacer click derecho sobre el archivo y abrirlo con un editor de texto. Cuando hayas terminado de modificarlo solo vuelve a repetir los pasos anteriores en orden.

¿Como enlazo un documento CSS con un archivo HTML?

Para enlazar un documento CSS con un archivo HTML hay que hacer uso de la etiqueta <link> y dentro de ella añadir los atributos rel="" con el valor stylesheet y el atributo href="" con el valor ruta siendo ruta la ruta hacia tu archivo. Esta etiqueta irá alojada entre las etiquetas <head> y </head>

Ejemplo 1: El documento CSS se encuentra en la misma carpeta que el archivo HTML 


    <head>
        <link rel="stylesheet" href="./style.css">
    </head>

Ejemplo 2: El documento CSS se encuentra dentro de una carpeta que está alojada en la misma ruta que el archivo HTML. 


    <head>
        <link rel="stylesheet" href="./nombre_carpeta/style.css">
    </head>

Ejemplo 3: El documento CSS se encuentra dentro de una carpeta que no está alojada en la misma ruta que el archivo HTML. 


    <head>
        <link rel="stylesheet" href="../../nombre_carpeta/style.css">
    </head>

NOTA Los archivos CSS no necesitan ningún elemento por defecto para ejecutarse.

NOTA No olvides de enlazar tu documento CSS con el archivo HTML al que vas a aplicar los estilos.

Tres propiedades CSS muy utilizadas

background-color: Esta propiedad como su nombre indica sirve para cambiar el color de fondo de un elemento HTML. El valor mas utilizado es el hexadecimal y el rgb().

  • Por ejemplo si utilizamos la propiedad con un valor en hexadecimal se vería algo así: 
    
    background-color: #ff0000;
  • Mientras que si usamos un valor rgb(0,0,0) se vería algo así: 
    
    background-color: rgb(255, 0, 0);

color. La propiedad color nos permite cambiar el color del texto de un elemento HTML. Al igual que con background-color los valores más utilizados son en hexadecimal o rgb().

  • Por ejemplo si utilizamos la propiedad con un valor en hexadecimal se vería algo así: 
    
    color: #00ffff;
  • Mientras que si usamos un valor rgb(0,0,0) se vería algo así: 
    
    color: rgb(255, 0, 255);

margin, puede ser considerada una de las propiedades más importantes, ya que nos sirve para alinear un elemento, ademas de darle un margen concreto hacia un lado especifico. Entre todas las combinaciones posibles, las más normales y utilizadas son:

  • Esta combinación permitirá que el elemento se posicione en el centro de la caja que lo contiene. 
    
    margin: 0 auto;
  • Esta combinación eliminará todo tipo de margen que pueda tener el elemento. 
        
    margin: 0;

NOTA Todos los elementos HTML tienen algún estilo aplicados por defecto. Cada navegador aplicará unos estilos independientes por defecto. Por lo que a la hora de diseñar una página o documento hay que pensar si se van a sobreescribir o dejar los que están por defecto.

NOTA Cuando cambies el color de fondo del elemento body veras que queda un recuadro en blanco, eso es el margen y para eliminarlo deberás de hacer uso de uno de los ejemplos facilitados arriba.
